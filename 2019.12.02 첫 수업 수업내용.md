### 데이터 타입

- 숫자데이터 - 정수(byte [1바이트 크기의 숫자 표현 -128~127], short(2byte), int(4byte), long(8byte)), 교재 29p

- 단일 인용부호로 두개 이상의 문자데이터는 만들 수 없다.

  '가' = ok , '가나다' = X -> "가나다"

  1			1.0				'1'				"1"

  정수		실수			문자			문자열

- '1'은 문자형의 1인데 아스키코드 상으로 49를 의미한다(아스키 코드 참조)

  문자열이 존재할 경우 '+'는 너미지 숫자까지 문자열로 바꾸어 인식한다. 따라서 "1"+"1"="11"

```java
System.out.println(1+1);	//2
System.out.println(1.0+1);	//1.0+1 ->1.0+1.0 ->2.0
System.out.println('1'+1);	//1이라는 문자의 값 =>아스키코드상 49라고 인식한다.
System.out.println("1"+1);	//앞이 문자열일 경우 '+'는 뒤의 숫자까지 문자열로 바꾸어 인식한다. 따라서 "1"+"1"="11"
```

### 연산자

- 기능 : 산술연산자, 비교연산자, 논리연산자, 조건연산자, 대입연산자

- 사용되는 항(피연산자, 연산에 사용되는 데이터)의 갯수 => 

  - 단항연산자 : ++num(num=num+1)

  - 이항연산자 : 항1 연산자 항2

  - 삼항연산자 : 항1(true or false) ? 항2 : 항3   //항1이 참일 경우 항2를 수행하고 거짓일 경우 항3을 수행한다

    ex) int bigNumber = num1>num2 ? num1 : num2;

    num1이 num2보다 크다면 num1 값을 넣고 아니라면 num2 값을 넣어라



​	++, --, +, -, !, ~, (타입명), *, /, %(나머지), ==(동일한가? 참/거짓), !=(동일하지 않나?)

	>, <, >=, <=, >>, >>>, <<, <<<, &, |, ^ ,&&(논리 and), ||(논리 or)
	>
	>항1 ? 항2 : 항3(조건 연산자), =, +=, -=, *=, /=, instanceof
	>
	>

```java
int su = 10; //su 변수의 값을 1 증가시키시오
su = su+1;
++su;
su +=1;

int su=10;	//su 변수의 값을 3 증가시키시오
su = su+3;
su +=3;
++su; ++su; ++su;
```
